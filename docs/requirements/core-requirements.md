# 漢字ジャストワン（仮）要件定義 v1

## 概要

1台のiOSデバイスを複数人で回して遊ぶ、パーティゲーム。回答者以外の参加者が、お題に関するヒントを漢字で出し合い、回答者がお題を当てる。他の出題者と漢字が被ると無効になるため、被らない漢字を選ぶ駆け引きが core loop。

## 対象プラットフォーム・デバイス

- iOS アプリ
- 1台のデバイスを回して遊ぶローカルパスプレイ専用（v1）
- オンライン対戦・複数端末連携は将来検討事項。実装制約が大きい場合は見送る

## ゲームルール

### 1ターン（1つのお題）の流れ

1. **お題決定**：組み込みお題プールからランダム抽選、または「自作お題専用モード」の場合は自分たちが登録したリストから抽選
2. **お題公開フェーズ**：回答者以外の全員に、順番に端末を回して見せる。見せる前に「〇〇さんですか？」と確認する。全員が見終わったら、誰でもボタン一つでお題を引き直す（スキップ）ことができる。回数制限やトリガー権限の制約は設けない
3. **ヒント入力フェーズ**：ヒント出題者に順に端末を回し、都度「〇〇さんですか？」の確認を挟んで各自が漢字を入力する（基本2文字。設定で1人あたりの文字数を変更可）。各フェーズに任意で制限時間（タイマー）を設定できる
4. **自動重複削除**：全員の入力が終わったら、**1文字単位**で重複している文字をシステムが自動的に取り除く（例：Aさんの「山川」とBさんの「山風」→ 双方から「山」が消え、「川」「風」だけが残る）
5. **グループ確認フェーズ**：ヒント出題者全員で、生き残った文字を確認する。
   - 同一人物が出した漢字は**グルーピング表示**する（「誰が」は分からないが「同じ人物の組」だと分かる形）。これは「同一人物の2文字で実在の言葉を作っていないか」を出題者自身がチェックできるようにするため
   - グループの表示順は**毎回ランダムにシャッフル**する（提出順・端末を回した順とは無関係にし、位置から誰の漢字か推測できないようにする）
   - 自動重複削除で消えた文字は、このフェーズでは**完全に非表示**（生き残った文字のみが見える）
   - ルール違反（お題に含まれる漢字・和訳した漢字やその一部の使用、同一人物の2文字で実在の言葉を作っている、等）に気づいた場合、**全員一致で手動削除**できる。この違反判定自体はアプリは行わず、人力で行う
6. **回答フェーズ**：回答者に端末を渡し、生き残った漢字を**フラット**に見せる（グルーピングなし、順不同）。回答者がお題を入力する
7. **答え合わせ**：ヒント出題者に見せて正誤判定する
   - **正解**の場合：回答者以外の全員に、ヒント出題者の人数分の得点を付与。さらに、自動削除＋手動削除の**合計削除数**の分だけボーナス加点
   - **ギブアップ（正解できない）**の場合：全員0点。以降の採点フェーズ・全公開フェーズはスキップ
8. **ヒント採点フェーズ**（正解時のみ）：回答者が生き残った漢字を「役に立った順」に順位付けする。生き残った漢字がN個ある場合、1位から順にN, N-1, ..., 1点を各出題者に付与する（順位式）
9. **全公開フェーズ**：誰がどの漢字を出したか、自動・手動で削除された漢字も含めて全て公開し、このお題を終了する

### ラウンドとゲームの構造

- 参加者全員が1回ずつ回答者を経験する1周を**1ラウンド**と呼ぶ
- ラウンド数（周回数）はゲーム開始時に選択できる
- 回答者の決め方は「固定ローテーション」「毎回選択（ランダム／任意選択）」のどちらのモードも用意する
- 合計得点で勝敗を決める

### 設定項目

- 参加人数：2〜8人（推奨3〜6人）
- 1人あたりの漢字文字数：デフォルト2文字。変更可能で、変更した場合も「その文字数で実在する言葉を作らない」というルールは維持する
- ラウンド数
- 難易度：易・普通・難の3段階。**ラウンド単位**で固定し、ラウンド中に難易度は変わらない（ラウンドごとに難易度を変えることは可能）
- タイマー：任意設定（デフォルト無制限）

## 参加者管理

- 参加者（プレイヤー）リストは端末に保存する
- 次回以降のプレイでは、リストから選択するだけで参加できる（毎回名前を手入力する必要はない）

## お題（コンテンツ）

### 組み込みお題

- 3段階の難易度 × カテゴリで、合計数百問規模を想定
- 制作方法：AI（LLM）で下書きを生成し、人間が目視チェック・修正して確定させる
- 難易度の具体的な線引き基準（言葉の抽象度・使用頻度など）はコンテンツ制作時に確定する（TBD）

### オリジナルお題（自作モード）

- ユーザーが自由テキストでお題を登録できる（バリデーションなし）
- 組み込みお題の抽選プールには混ぜず、「自作お題専用モード」として別立てで提供する（自分たちが登録したリストからのみ出題される）

## 履歴・統計

- v1から簡易な履歴・統計画面を用意する
  - 過去の対戦履歴（日時・参加者・得点など）
  - 個人の集計データ（例：有効な漢字をどれだけ出せているか、など）
- 裏側では「誰が・どのお題に対して・どの漢字を出したか」を記録し、将来的なAI分析（その漢字で正解できたかの予測、性格診断など）のためのデータ基盤とする。ただし**AI分析機能自体はv1のスコープ外**（将来バージョンで実装）

## マネタイズ

- 無料版：お題数・参加人数上限・ラウンド数上限を制限し、広告を表示する
- 買い切り：単一の「全解除」商品1種類のみ（広告除去＋各種制限解除）。DLC的な追加お題パックの個別課金などは設けない
- オリジナルお題（自作モード）は買い切り限定機能
- 無料版の具体的な制限値（お題数・人数上限・ラウンド数上限）、広告の種類・表示タイミングは未定（TBD）

## v1スコープ外

- AI分析機能（正誤予測・性格診断）
- Game Center連携（実績・リーダーボード）
- オンライン対戦・複数端末連携
- 買い切り以外の追加課金（お題パック等の個別販売）
